# Security Policy

If you discover a security vulnerability in JARVIS, please help us fix it
responsibly.

- Preferred contact: **security@yourdomain.example** (replace with your security contact)
- Alternatively, open an issue and add the label `security` (do NOT include sensitive details publicly)

What we will do:
- Acknowledge your report within 72 hours
- Inform you of the mitigation timeline
- Coordinate disclosure and credit as appropriate

**Do not** post exploit details or proof-of-concept to public issue trackers.
